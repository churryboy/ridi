# RIDI Design System Documentation

This design system is reverse-engineered from the RIDI mobile application screenshots.

## 1. Design Tokens

### 1.1 Colors

| Swatch | Code | Name |
|--------|------|------|
| 🟦 | `#1F8CE6` | Primary Blue |
| ⬛ | `#333333` | Text Primary |
| ⬜ | `#F2F4F5` | Secondary BG |

### 1.2 Typography

- **Headings:** Bold, Sans-serif (20px - 24px)
- **Body:** Regular, Sans-serif (14px - 16px)
- **Metadata:** Light/Regular (12px)

## 2. UI Components

- **Navigation Bar:** Minimalist, icon-based actions.
- **Search Bar:** Pill-shaped, light gray background.
- **Filter Chips:** Rounded edges, blue for active state.
- **Cards:** Vertical book covers or horizontal banner carousels.
